## 8. Roadmap

### 🗺️ Learning Path

```
Month 1-2: Foundations
├── Variables, Data Types, Operators
├── Conditionals, Loops
├── Functions, Scope
├── Arrays, Objects
└── DOM Basics

Month 3-4: Intermediate
├── ES6+ Features
├── Array Methods (map, filter, reduce)
├── Async JavaScript (Promises, async/await)
├── Events, Event Delegation
├── LocalStorage, APIs
└── Build 2-3 Projects

Month 5-6: Advanced
├── Closures, Prototypes
├── this keyword, bind/call/apply
├── Modules, Webpack/Vite
├── Testing (Jest, Vitest)
├── TypeScript Basics
└── Build Portfolio

Month 7+: Specialization
├── React/Vue/Angular
├── Node.js Backend
├── Full-stack Projects
└── Job Preparation
```

### 📋 Skill Checklist

#### Beginner (Weeks 1-8)
- [ ] `let`, `const`, `var` (know when to use each)
- [ ] Data types (string, number, boolean, null, undefined)
- [ ] Operators (==, ===, !=, !==, &&, ||, ??)
- [ ] Conditionals (if/else, switch, ternary)
- [ ] Loops (for, while, for...of, for...in)
- [ ] Functions (declaration, expression, arrow)
- [ ] Arrays (push, pop, map, filter, reduce)
- [ ] Objects (create, access, spread)
- [ ] DOM manipulation (querySelector, createElement)
- [ ] Events (addEventListener, event object)

#### Intermediate (Weeks 9-16)
- [ ] Destructuring (array, object)
- [ ] Rest/Spread operator
- [ ] Template literals
- [ ] Modules (import/export)
- [ ] Promises, async/await
- [ ] Fetch API
- [ ] Error handling (try/catch)
- [ ] Closures
- [ ] this keyword
- [ ] Classes, inheritance

#### Advanced (Weeks 17-24)
- [ ] Prototypes, prototype chain
- [ ] bind, call, apply
- [ ] Event loop, microtasks
- [ ] Generators, Iterators
- [ ] Proxies, Symbols
- [ ] Web Workers
- [ ] Service Workers
- [ ] Design patterns
- [ ] Performance optimization
- [ ] Testing strategies

## 🎯 Quick Start Guide

### For Absolute Beginners

```
Week 1:
1. Read: javascript.info - Introduction
2. Practice: Variables, Data Types exercises
3. Build: Simple calculator

Week 2:
1. Read: MDN - Conditionals & Loops
2. Practice: FizzBuzz, Prime numbers
3. Build: Number guessing game

Week 3:
1. Read: javascript.info - Functions
2. Practice: Function exercises
3. Build: To-do list (console version)

Week 4:
1. Read: MDN - Arrays & Objects
2. Practice: Array methods
3. Build: Contact list
```

### For Developers Switching to JS

```
Week 1:
1. Skim: JavaScript.info basics
2. Focus: ES6+ features
3. Build: Refactor old code to modern JS

Week 2:
1. Read: You Don't Know JS (Scope & Closures)
2. Practice: Codewars kata
3. Build: API integration project

Week 3:
1. Read: Async JavaScript deep dive
2. Practice: Promise exercises
3. Build: Full CRUD app with localStorage
```
