## 🎯 Skill Checklist

### Beginner (Weeks 1-8)

- [ ] `let`, `const`, `var` (know when to use each)
- [ ] Data types (string, number, boolean, null, undefined, symbol, bigint)
- [ ] Operators (`==` vs `===`, `??`, `?.`)
- [ ] Conditionals (if/else, switch, ternary)
- [ ] Loops (for, while, for...of, for...in)
- [ ] Functions (declaration, expression, arrow)
- [ ] Arrays (push, pop, map, filter, reduce, find, some, every)
- [ ] Objects (create, access, spread, destructuring)
- [ ] DOM manipulation (querySelector, createElement, classList)
- [ ] Events (addEventListener, event object, delegation)

### Intermediate (Weeks 9-16)

- [ ] Closures
- [ ] this keyword (bind, call, apply)
- [ ] Prototypes, inheritance
- [ ] Classes (extends, super, static)
- [ ] Modules (import/export)
- [ ] Promises, async/await
- [ ] Fetch API, error handling
- [ ] LocalStorage, SessionStorage
- [ ] Event Loop (microtasks, macrotasks)
- [ ] Debounce, Throttle

### Advanced (Weeks 17-24)

- [ ] Generators, Iterators
- [ ] Proxies, Symbols
- [ ] Web Workers
- [ ] Service Workers
- [ ] Design Patterns (Module, Singleton, Observer)
- [ ] Performance Optimization
- [ ] Testing (Jest, Vitest)
- [ ] TypeScript Basics
- [ ] Build Tools (Vite, Webpack)
- [ ] CI/CD Basics

## 📝 Quick Reference

### Variables

```javascript
const PI = 3.14159;  // Hằng số
let count = 0;       // Biến
// Tránh dùng var
```

### Functions

```javascript
// Arrow function
const add = (a, b) => a + b;

// Default params
function greet(name = "Anonymous") {
  return `Hello, ${name}!`;
}
```

### Arrays

```javascript
arr.map(x => x * 2);        // Transform
arr.filter(x => x > 5);     // Filter
arr.reduce((a, b) => a + b, 0); // Reduce
arr.find(x => x > 5);       // Find first
arr.some(x => x > 5);       // Check any
arr.every(x => x > 0);      // Check all
```

### Async

```javascript
async function fetchData() {
  try {
    const res = await fetch("/api/data");
    return await res.json();
  } catch (error) {
    console.error(error);
  }
}
```
